Excel・スプレッドシートからまとめて登録

すでにExcelやGoogleスプレッドシートで管理表を作っている場合、1行ずつ入力し直す必要はありません。1行目（見出し行）の項目名を下記の名前に合わせるだけで、列の並び順はそのままで取り込めます。
① 管理表の1行目（見出し）を「対応している列名」に書き換える（すでに近い名前なら不要）
② Excel／スプレッドシートで「ファイル」→「ダウンロード」または「名前を付けて保存」→ 「CSV（カンマ区切り）」形式で保存する
③ 「CSVファイルを選択」ボタンから、保存したファイルを選ぶと自動で取り込まれます

新しく管理表を作る場合は、「CSVテンプレートをダウンロード」した見本ファイルに直接入力してください。

対応している列名（1行目に1つずつ、順不同でOK）:
商品名（品名／商品／タイトルでも可） / 仕入先（仕入れ先／購入先／店舗でも可） / プラットフォーム（販売先／出品先でも可） / ステータス（在庫・出品中・売却済みのいずれかを入力） / 仕入日 / 仕入額（原価でも可） / 販売日（売却日でも可） / 販売額（売価でも可） / 手数料 / 送料 / 獲得ポイント（ポイントでも可） / メモ（詳細／備考でも可）。
「商品名」列以外は無くても構いません。取り込んだデータは既存のデータに追加されます（上書きされません）。

データのエクスポート／インポート

全データはブラウザの localStorage にのみ保存されます。他の端末に移す場合や、バックアップを取る場合は JSON ファイルとして書き出してください。
